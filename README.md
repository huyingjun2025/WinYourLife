# WinYourLife 发布说明（中文）
WinYourLife 是一个把“做任务”变成“玩系统”的个人成长小工具。
你可以自己定义任务、奖励、卡池和活动规则，把每天要做的事情变成一个有反馈、有成就感的循环：完成任务拿金币，用金币兑换奖励或参与抽卡，把努力变成看得见的进度。
整体轻量、可自定义、长期可坚持，也支持记录、导入导出和报告导出，获取成就感与反馈。

## 快速使用

1. 下载最新 `WinYourLife_x64-setup.exe`
2. 双击安装
3. 从桌面快捷方式启动
